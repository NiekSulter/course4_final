About this project

Een groot probleem binnen de champignonteelt is echter dat de oogst van champignons sterk kan variëren.
Ook micro-organismen die het pasteurisatie proces in de compost overleefd hebben kunnen een negatieve of juist een positieve invloed hebben op de teelt.
Deze micro-organismen vormen in de compost een microflora. Het DNA van deze is gesequenced en vervolgens geblast om te kijken welke eiwitten deze produceren.

Resultaten:
Hier staan de blast resultaten van het DNA uit de champignoncompost die door het HAN Biocenter gesequenced zijn.
Je kan filteren op proteïne naam, functie en taxonomy. E-value is hoe betroubaar de blast reasultaat is, hoe kleiner hoe beter.
Identity is hoeveel de sequenties overeenkomen. Voor prokaryoten is boven de 40% goed.
Query cover is het percentage van de query sequentie dat de referentie sequentie overlapt.
Elke accessiecode heeft een hyperlink die naar de corresponderende NCBI pagina verwijst.

Blast:
Hier kan de gebruiker een eigen sequentie via de website blasten. Optioneel kan de gebruiker een email adres achterlaten om een notificatie te krijgen wanneer de blast voltooid is.

User blast results:
Hier zijn de resultaten van een door de gebruiker uitgevoerde blast te vinden. Als de gebruiker een ID meegeeft zullen alleen de resultaten van die specifieke job opgehaald worden.
Als de gebruiker geen ID meegeeft zullen alle resultaten in de database opgehaald worden.
